Getting Started: Remote Control Demo

 

The remote control demo runs on the Remote Control Demo Board (DM240315). This board is not currently available at the time of this software release. The expected public availability date for this board is CQ2 2012. When available, the board user’s guide detailing the steps necessary in running this demo will be available from www.microchip.com/RemoteControl.

 

The Remote Control Demo Board is a platform that demonstrates the hardware and software integration of various different technologies, including Graphics, mTouch keys, USB, and Wireless RF4CE, controlled by a single PIC24FJ256DA210 microcontroller. The 16-bit microcontroller integrates a graphics controller, CTMU and USB OTG modules essential for integrated human interface applications. The wireless module is the MRF24J40MA, 2.4 GHz 802.15.4 transceiver.
