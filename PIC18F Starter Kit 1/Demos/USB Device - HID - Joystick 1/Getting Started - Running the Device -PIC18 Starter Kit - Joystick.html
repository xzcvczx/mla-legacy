Required Hardware:

 To run this project, you will need PIC18 Starter Kit

 

 

Configuring the Hardware:

 

This section describes how to set up the various configurations of hardware to run this demo.

 

PIC18 Starter Kit

            1) Short JP3 to allow the board to be powered from the USB.

           

 

 

 

Firmware:

To run this project, you will need to load the corresponding firmware into the devices.  There are two methods available for loading the demos: Precompiled demos and source code projects.

 

 

Running the Demo:

This demo uses the PIC18 Starter kit as a USB Joystick.  To test the working of the joystick, Open Control Panel -> Game Controllers. Below ‘Controller’ column select the device ‘Joystick Demo’ and click on Properties Tab.  Verify the working of Joystick using the properties.

 

            The following controls are implemented in the Joystick Demo

           

1) X-Axis / Y-Axis -> The X and Y axis are controlled using an Accelerometer on the board (BMA150). To verify X and Y axis hold the PIC18 Starter kit in a manner so that USB connector faces towards you. Tilt the board toward right and left to change the X-Axis position.  Tilt the board upward and downward to change the Y-axis position.  

2) Z-Axis -> The Z-axis is controlled using the mTouch slider. Slide towards bottom and top to change the Z-axis position.

 

3) Z-rotation -> The Z-rotation is controlled using the potentiometer. Rotate the potentiometer to change the Z-Rotation.

 

4) Button 1 and 2 -> The button 1 and 2 are controlled using the mTouch switches L and R. Press and hold the button to turn on the button.
