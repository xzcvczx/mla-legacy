The PIC32MX MSD Host Boot loader support in MLA will be deprecated. 
It is recommended to use PIC32MX MSD Host Bootloader which is described in Microchip Application Note AN1388. 
The application note can be downloaded from the following link:
http://www.microchip.com/stellent/idcplg?IdcService=SS_GET_PAGE&nodeId=1824&appnote=en554836
