This file contains the information related to the mTouch Capacitive Touch Software Library 
used for AN1317_Demo. This demo code includes the conducted noise immune code using CTMU for capacitive touch application. For the details of implementation refer to AN1317 on the website www.microchip.com

*******************************************************************************
    	mTouch Capacitive Touch Library - AN1317 demo
*******************************************************************************
This demo is developed based on the AN1317 application note. The code for this demo is self contained in the project marked as mTouchCap_AN1317 in the following directory
\\Microchip Solutions\mTouchCapDemos\PIC24F_Demos\mTouchCap_AN1317

Please note that this release of AN1317 does not use the mTouch library APIs.
